📱 30 App - Download Page

This repository hosts the landing page for downloading the Android APK of the 30 Task Manager App.

🚀 Live Demo

👉 https://30-landing.vercel.app

📦 Contents
index.html → Landing page UI
30-app.apk → Android application file
📥 How it works
Users visit the website
Click Download APK
Install app directly on Android device
⚙️ Deployment

Hosted using:

Vercel (Auto-deploy via GitHub)
🔄 Updating APK

To update the app:

Replace 30-app.apk with new version
Run:
git add .
git commit -m "update apk"
git push
Vercel auto-deploys 🚀
